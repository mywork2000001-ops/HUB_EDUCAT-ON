§1. Bucağın radian ölçüsü

Radian

Uzunluğu radiusa bərabər olan yayın mərkəz bucağına 1 radian deyilir.

Dərəcə ↔ Radian çevrilməsi
α° = α · π / 180 (radian)
α rad = α · 180 / π (dərəcə)

Əsas bucaqlar

Dərəcə 0° 30° 45° 60° 90° 180° 360°
Radian 0 π/6 π/4 π/3 π/2 π 2π

§2. Trigonometrik nöqtə — vahid çevrə

Vahid çevrə

Mərkəzi koordinatların başlanğıcı O(0;0), radiusu 1 olan çevrəyə vahid çevrə deyilir.

α bucağına uyğun çevrə üzərindəki M(x; y) nöqtəsi — trigonometrik nöqtədir.

Tərif: sin və cos
sin α = y (nöqtənin ordinatı)
cos α = x (nöqtənin absisi)
M(cos α; sin α) — vahid çevrə üzərindəki nöqtə

§3. Tangens və kotangens

Tərif
tg α = sin α / cos α   (cos α ≠ 0)
ctg α = cos α / sin α   (sin α ≠ 0)

Tərif oblastları

  • tg α: D = {α ∈ ℝ : α ≠ π/2 + πn, n ∈ ℤ}, yəni α ≠ 90° + 180°·n
  • ctg α: D = {α ∈ ℝ : α ≠ πn, n ∈ ℤ}, yəni α ≠ 180°·n
Əsas əlaqə
tg α · ctg α = 1
Hər iki birlikdə tərif edilən α üçün keçərlidir

§4. Əsas trigonometrik eynilik

Pifaqor eyniliyi
sin² α + cos² α = 1
Vahid çevrə üzərindəki nöqtə üçün: x² + y² = 1
Digər eyniliklər
1 + tg² α = 1 / cos² α   (cos α ≠ 0)
1 + ctg² α = 1 / sin² α   (sin α ≠ 0)

Tətbiq nümunəsi

sin α = 3/5 verilmişdir, α I rübdədir. cos α = ?

cos² α = 1 − sin² α = 1 − 9/25 = 16/25 → cos α = 4/5 (I rüb: cos > 0)

tg α = sin α / cos α = (3/5)/(4/5) = 3/4

§5. Xüsusi bucaqların dəyərləri

Bucaq 0° 30° (π/6) 45° (π/4) 60° (π/3) 90° (π/2)
sin 0 1/2 √2/2 √3/2 1
cos 1 √3/2 √2/2 1/2 0
tg 0 1/√3 1 √3 —
ctg — √3 1 1/√3 0
Yadda saxlama üsulu — "pillə" qaydası

sin dəyərləri 0° → 90°: 0, √1/2, √2/2, √3/2, √4/2 = 0, 1/2, √2/2, √3/2, 1

cos dəyərləri — sinusun əksinə: 1, √3/2, √2/2, 1/2, 0

§7. İndirasiya düsturları

π/2, π, 3π/2, 2π kəmiyyətlərinin əlavə/çıxılması zamanı:

Düstur sin cos
−α −sin α cos α
π − α sin α −cos α
π + α −sin α −cos α
2π − α −sin α cos α
π/2 − α cos α sin α
π/2 + α cos α −sin α
Qayda

§8. Sinusun qrafiki — y = sin x

Xassələr

• Tərif oblastı: D(sin) = ℝ  • Dəyərlər: [−1; 1]
• Dövr: T = 2π (360°)  • Tək funksiya: sin(−x) = −sin x

SLAYD 9 / 15

§9. Kosinusun qrafiki — y = cos x

Xassələr

• Tərif oblastı: D(cos) = ℝ  • Dəyərlər: [−1; 1]
• Dövr: T = 2π (360°)  • Cüt funksiya: cos(−x) = cos x

Qrafik y = sin x qrafikini π/2 (90°) sola sürüşdürərək alınar: cos x = sin(x + π/2)

§10. Tangensin qrafiki — y = tg x

Xassələr

• Tərif oblastı: α ≠ π/2 + πn  • Dəyərlər: (−∞; +∞)
• Dövr: T = π (180°)  • Tək funksiya: tg(−x) = −tg x
• Şaquli asimptotlar: x = π/2 + πn

§12. Toplama düsturları

Sin və cos cəmi/fərqi
sin(α ± β) = sin α · cos β ± cos α · sin β
cos(α ± β) = cos α · cos β ∓ sin α · sin β
İkiqat bucaq düsturları
sin 2α = 2 · sin α · cos α
cos 2α = cos² α − sin² α = 1 − 2sin² α = 2cos² α − 1
Yarımqat düsturu
sin² α = (1 − cos 2α) / 2
cos² α = (1 + cos 2α) / 2

§13. Ən sadə trigonometrik tənliklər

sin x = a   (|a| ≤ 1)
x = (−1)ⁿ · arcsin a + π · n,   n ∈ ℤ
Xüsusi hal: sin x = 0 → x = πn; sin x = 1 → x = π/2 + 2πn
cos x = a   (|a| ≤ 1)
x = ± arccos a + 2π · n,   n ∈ ℤ
Xüsusi hal: cos x = 0 → x = π/2 + πn; cos x = 1 → x = 2πn
tg x = a
x = arctan a + π · n,   n ∈ ℤ
Xüsusi hal: tg x = 0 → x = πn; tg x = 1 → x = π/4 + πn

§14. Trigonometrik bərabərsizliklər

Trigonometrik bərabərsizlikləri həll etmək üçün vahid çevrə istifadə edilir.

sin x > a həlli

  • a ≥ 1: həll yoxdur
  • a ≤ −1: bütün x həlldir: x ∈ ℝ
  • −1 < a < 1: vahid çevrədə sin x > a olan yay

Nümunə: sin x > 1/2

arcsin(1/2) = π/6. Həll: π/6 + 2πn < x < 5π/6 + 2πn, n ∈ ℤ
