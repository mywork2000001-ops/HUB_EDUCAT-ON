§1. Loqarifm — Təkrar

Tərif

$a$ əsaslı $b$ ədədinin loqarifmi — $a^x = b$ tənliyinin həllidir:

Tərif
$$\log_a b = x \iff a^x = b$$
Şərt: $a > 0,\; a \neq 1,\; b > 0$

Xüsusi loqarifmlər

YazılışOxunuşƏsas
$\lg x$desimal loq10
$\ln x$natural loq$e \approx 2.718$
$\log_2 x$ikili loq2

Əsas loqarifm eyniliklər

  • $\log_a(mn) = \log_a m + \log_a n$
  • $\log_a\!\dfrac{m}{n} = \log_a m - \log_a n$
  • $\log_a m^k = k \cdot \log_a m$
  • $\log_a a = 1$  ·  $\log_a 1 = 0$  ·  $a^{\log_a b} = b$

§2. ODZ — Tənliyin Tərif Oblastı

Diqqət — ODZ hər zaman yoxlanılmalıdır!

Loqarifmik tənlikdə hər loqarifmin arqumenti müsbət olmalıdır. Loqarifmin əsası isə $a > 0,\; a \neq 1$ şərtini ödəməlidir.

ODZ şərti
$$\log_a f(x) \text{ mövcuddur} \iff f(x) > 0$$

Nümunə

$\log_3(x - 2)$ üçün ODZ: $x - 2 > 0 \Rightarrow x > 2$

$\log_5(x^2 - 4)$ üçün ODZ: $x^2 - 4 > 0 \Rightarrow x < -2$ və ya $x > 2$

Həll ardıcıllığı

1. ODZ-i tapın (loqarifm arqumentlərini $> 0$ ilə bərabərləyin)

2. Tənliyi həll edin

3. Alınan kökü ODZ-ə yoxlayın → uyğun olmayanları silin

§3. Növ I: $\log_a f(x) = b$

Əsas növ — rəqəmə bərabər
$$\log_a f(x) = b \iff f(x) = a^b \quad (f(x) > 0)$$

Nümunə 1

$\log_3(2x - 1) = 2$

Həll

ODZ: $2x - 1 > 0 \Rightarrow x > \tfrac{1}{2}$

$2x - 1 = 3^2 = 9$

$2x = 10 \Rightarrow x = 5$

Yoxlama: $x = 5 > \tfrac{1}{2}$ ✓ → Cavab: $x = 5$

Nümunə 2

$\log_2(x + 1) = 3$

Həll

ODZ: $x + 1 > 0 \Rightarrow x > -1$

$x + 1 = 2^3 = 8 \Rightarrow x = 7$ ✓

§4. Növ II: $\log_a f(x) = \log_a g(x)$

Eyni əsaslı loqarifmlər bərabərdir
$$\log_a f(x) = \log_a g(x) \iff f(x) = g(x)$$
ODZ: həm $f(x) > 0$, həm $g(x) > 0$ olmalıdır

Nümunə

$\log_5(x + 3) = \log_5(2x - 1)$

Həll

ODZ: $x + 3 > 0$ → $x > -3$;   $2x - 1 > 0$ → $x > \tfrac{1}{2}$

Yəni ODZ: $x > \tfrac{1}{2}$

$x + 3 = 2x - 1 \Rightarrow x = 4$

Yoxlama: $4 > \tfrac{1}{2}$ ✓ → Cavab: $x = 4$

Səhv etmə

ODZ yoxlamasını atlayıb yalnız $f(x) = g(x)$-dən gələn kökü qəbul etmə — o kök ODZ-ə uymaya bilər!

§5. Loqarifmlərin Cəmi/Fərqi

Eyni əsaslı loqarifmlərin birləşdirilməsi
$$\log_a f + \log_a g = \log_a(fg)$$ $$\log_a f - \log_a g = \log_a\!\frac{f}{g}$$

Nümunə

$\log_2 x + \log_2(x - 2) = 3$

Həll

ODZ: $x > 0$ və $x - 2 > 0 \Rightarrow x > 2$

$\log_2[x(x-2)] = 3$

$x(x - 2) = 2^3 = 8$

$x^2 - 2x - 8 = 0 \Rightarrow (x - 4)(x + 2) = 0$

$x = 4$ ya da $x = -2$

ODZ yoxlaması: $x = -2 < 2$ — uyğun deyil. $x = 4 > 2$ ✓

Cavab: $x = 4$

§6. Əvəzetmə üsulu — $t = \log_a x$

Əvəzetmə

$t = \log_a x$ qoyaraq tənliyi $t$-ə görə kvadrat (və ya digər) tənliyə çevir, həll et, sonra geri qayıt.

Nümunə

$(\log_3 x)^2 - 5\log_3 x + 6 = 0$

Həll

ODZ: $x > 0$

$t = \log_3 x$ qoyaq:

$t^2 - 5t + 6 = 0$

$(t - 2)(t - 3) = 0 \Rightarrow t = 2$ ya da $t = 3$

$t = 2$: $\log_3 x = 2 \Rightarrow x = 9$ ✓

$t = 3$: $\log_3 x = 3 \Rightarrow x = 27$ ✓

Cavab: $x = 9;\; x = 27$

§7. Əsas Dəyişmə Düsturu

Əsas dəyişmə
$$\log_a b = \frac{\log_c b}{\log_c a} = \frac{\ln b}{\ln a}$$
İstənilən müsbət $c \neq 1$ üçün keçərlidir

Faydalı xüsusi hal

Xüsusi hal
$$\log_a b = \frac{1}{\log_b a}$$

Müxtəlif əsaslı tənlik nümunəsi

$\log_2 x + \log_4 x = 3$

Həll — eyni əsasa çevirik

$\log_4 x = \dfrac{\log_2 x}{\log_2 4} = \dfrac{\log_2 x}{2}$

$t = \log_2 x$ qoyaq:   $t + \dfrac{t}{2} = 3 \Rightarrow \dfrac{3t}{2} = 3 \Rightarrow t = 2$

$\log_2 x = 2 \Rightarrow x = 4$ ✓

Cavab: $x = 4$

§8. Loqarifmik Bərabərsizliklər

Əsas fərq: əsasın işarəsi dəyişir!

$a > 1$ olduqda loqarifm artan funksiyadır → bərabərsizlik işarəsi saxlanır.
$0 < a < 1$ olduqda loqarifm azalan funksiyadır → bərabərsizlik işarəsi dəyişir.

$a > 1$ halı
$$\log_a f(x) > b \iff f(x) > a^b \quad \text{(ODZ ilə)}$$
$0 < a < 1$ halı
$$\log_a f(x) > b \iff 0 < f(x) < a^b$$

Nümunə ($a > 1$)

$\log_2(x - 1) > 3$

Həll

ODZ: $x - 1 > 0 \Rightarrow x > 1$

$a = 2 > 1$ → işarə saxlanır: $x - 1 > 2^3 = 8 \Rightarrow x > 9$

ODZ ilə kəsişim: $x > 9$. Cavab: $x \in (9; +\infty)$

Nümunə ($0 < a < 1$)

$\log_{0.5}(x + 2) > -1$

Həll

ODZ: $x + 2 > 0 \Rightarrow x > -2$

$a = 0.5 < 1$ → işarə dəyişir: $0 < x + 2 < (0.5)^{-1} = 2 \Rightarrow -2 < x < 0$

Cavab: $x \in (-2;\; 0)$

§9. Loq-Eksponent Qarışıq Tənliklər

Bəzi tənliklərdə həm loqarifm, həm de eksponent ifadə olur. Əvəzetmə üsulu çox işə yarayır.

Nümunə

$x^{\log_2 x} = 8x^2$

Həll — hər iki tərəfin $\log_2$-sini alaq

ODZ: $x > 0$

$\log_2 x \cdot \log_2 x = \log_2(8x^2) = 3 + 2\log_2 x$

$t = \log_2 x$ qoyaq:   $t^2 = 3 + 2t \Rightarrow t^2 - 2t - 3 = 0$

$(t - 3)(t + 1) = 0 \Rightarrow t = 3$ ya da $t = -1$

$t = 3$: $x = 8$;   $t = -1$: $x = \tfrac{1}{2}$

Cavab: $x = 8;\; x = \dfrac{1}{2}$

§10. Loqarifmik Tənliklər Sistemi

Sistem həlli üsulu

Hər tənliyi ayrı-ayrılıqda sadələşdir, sonra sistem kimi həll et. ODZ hər iki tənlik üçün eyni anda ödənilməlidir.

Nümunə

$$\begin{cases} \log_2 x + \log_2 y = 5 \\ \log_2 x - \log_2 y = 1 \end{cases}$$

Həll

ODZ: $x > 0$ və $y > 0$

Cəmləyək: $2\log_2 x = 6 \Rightarrow \log_2 x = 3 \Rightarrow x = 8$

Çıxaq: $2\log_2 y = 4 \Rightarrow \log_2 y = 2 \Rightarrow y = 4$

ODZ: $x = 8 > 0$, $y = 4 > 0$ ✓

Cavab: $x = 8,\; y = 4$
