§1. Vektor tərifi: istiqamətli parça, modul, sıfır vektor

Vektor

Uzunluğu (modulu) və istiqaməti olan kəmiyyətə vektor deyilir. Vektor ox işarəsi ilə: a⃗ və ya AB⃗ (A — başlanğıc, B — son).

  • Modul |a⃗|: vektorun uzunluğu (həmişə ≥ 0)
  • Sıfır vektor 0⃗: uzunluğu 0 olan vektor, istiqaməti müəyyən edilmir
  • Vahid vektor: modulu 1-ə bərabər olan vektor (e⃗ = a⃗/|a⃗|)
  • Bərabər vektorlar: eyni istiqamət VƏ eyni uzunluq — başlanğıc nöqtəsindən asılı deyil

§2. Vektorun koordinatları müstəvidə

Vektorun koordinatları
$$\vec{a} = (a_x,\; a_y)$$
$$\overrightarrow{AB} = (x_B - x_A,\; y_B - y_A)$$
A(x₁,y₁) və B(x₂,y₂) nöqtələri üçün: AB⃗ = (x₂−x₁, y₂−y₁)
Vektorun modulu (uzunluğu)
$$|\vec{a}| = \sqrt{a_x^2 + a_y^2}$$
Pifaqor teoremindən alınır

Nümunə

A(1, 2) və B(4, 6) nöqtələri üçün AB⃗ = ?

AB⃗ = (4−1, 6−2) = (3, 4)

|AB⃗| = √(3²+4²) = √(9+16) = √25 = 5

§3. Bərabər vektorlar və kollinear vektorlar

Bərabər vektorlar

İki vektor bərabərdir, əgər onların koordinatları eyni olsa: a⃗ = b⃗ ↔ (ax, ay) = (bx, by)

Kollinear (biistiqamətli) vektorlar

İki vektor kollineardır, əgər biri digərinin skalyar misliisə: a⃗ ∥ b⃗ ↔ a⃗ = k·b⃗ (k ∈ ℝ, k≠0)

Kollinear şərt koordinatlarla
$$\vec{a} \parallel \vec{b} \iff \frac{a_x}{b_x} = \frac{a_y}{b_y} \iff a_x \cdot b_y - a_y \cdot b_x = 0$$
Koordinatların "çarpaz hasil" sıfıra bərabərdir

Nümunə

a⃗ = (2, 3) və b⃗ = (4, 6). Kollinear mı?

2·6 − 3·4 = 12 − 12 = 0 → Bəli, kollinear (b⃗ = 2·a⃗)

a⃗ = (1, 2) və c⃗ = (3, 5). Kollinear mı?

1·5 − 2·3 = 5 − 6 = −1 ≠ 0 → Xeyr, kollinear deyil

§4. Vektorların toplanması və çıxılması

Vektorların toplanması
$$\vec{a} + \vec{b} = (a_x + b_x,\; a_y + b_y)$$
Parallelogram qaydasına görə: birinci vektorun sonuna ikincinin başlanğıcı qoyulur
Vektorların çıxılması
$$\vec{a} - \vec{b} = (a_x - b_x,\; a_y - b_y)$$

Nümunə

a⃗ = (3, −1), b⃗ = (2, 4). a⃗+b⃗ = (5, 3); a⃗−b⃗ = (1, −5)

§5. Skalyar vurma: k·a⃗

Skalyar vurma
$$k \cdot \vec{a} = (k \cdot a_x,\; k \cdot a_y)$$
k > 0 olduqda istiqamət qorunur; k < 0 olduqda istiqamət əksə dönür; k = 0 → sıfır vektor

Xassələr

  • $|k \cdot \vec{a}| = |k| \cdot |\vec{a}|$ — modul $|k|$ qat böyüyür/kiçilir
  • k = 1: $1 \cdot \vec{a} = \vec{a}$ (dəyişmir)
  • k = −1: $(-1) \cdot \vec{a} = -\vec{a}$ (yalnız istiqamət dəyişir)
  • $k(\vec{a}+\vec{b}) = k\vec{a} + k\vec{b}$ — paylanma qanunu

Nümunə

a⃗ = (2, 3). 3·a⃗ = (6, 9). |3·a⃗| = √(36+81) = √117 = 3√13.

Yoxlama: 3·|a⃗| = 3·√(4+9) = 3·√13. ✓

§6. Skalar (daxili) hasil

Skalar hasil — koordinatlarla
$$\vec{a} \cdot \vec{b} = a_x \cdot b_x + a_y \cdot b_y$$
Skalar hasil — modullar və bucaqla
$$\vec{a} \cdot \vec{b} = |\vec{a}| \cdot |\vec{b}| \cdot \cos\theta$$
θ — vektorlar arasındakı bucaq (0° ≤ θ ≤ 180°)

Xassələr

  • $\vec{a} \cdot \vec{a} = |\vec{a}|^2$ — vektorun özü ilə hasil modulu kvadrat verir
  • $\vec{a} \cdot \vec{b} = \vec{b} \cdot \vec{a}$ — dəyişdiricilik qanunu
  • $\vec{a} \cdot (\vec{b} + \vec{c}) = \vec{a}\cdot\vec{b} + \vec{a}\cdot\vec{c}$ — paylanma
  • Skalar hasil skalyar (ədəd) verir, vektor yox!

Nümunə

a⃗ = (1, 2), b⃗ = (3, 4). a⃗·b⃗ = 1·3 + 2·4 = 3 + 8 = 11

§7. Vektorlar arasındakı bucaq

Bucağın cosinus düsturu
$$\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}| \cdot |\vec{b}|} = \frac{a_x b_x + a_y b_y}{\sqrt{a_x^2+a_y^2} \cdot \sqrt{b_x^2+b_y^2}}$$
θ = arccos(...) — bucağı tapmaq üçün

Bucağa görə nəticə

cos θ dəyəriθMövqe
cos θ = 10°Eyni istiqamət
cos θ > 00° < θ < 90°Kəskin bucaq
cos θ = 090°Perpendikulyar
cos θ < 090° < θ < 180°Küt bucaq
cos θ = −1180°Əks istiqamət

Nümunə

a⃗ = (1, 0), b⃗ = (1, 1). |a⃗| = 1, |b⃗| = √2.

cos θ = (1·1 + 0·1)/(1·√2) = 1/√2 → θ = 45°

§8. Perpendikulyar və paralel şərtlər

Perpendikulyar şərt (θ = 90°)
$$\vec{a} \perp \vec{b} \iff \vec{a} \cdot \vec{b} = 0 \iff a_x b_x + a_y b_y = 0$$
cos 90° = 0 olduğundan skalar hasil sıfıra bərabərdir
Paralel şərt (θ = 0° və ya 180°)
$$\vec{a} \parallel \vec{b} \iff \vec{a} = k\vec{b} \iff a_x b_y - a_y b_x = 0$$
Koordinatların çarpaz hasili sıfıra bərabərdir

Nümunə 1: Perpendikulyar

a⃗ = (2, 3), b⃗ = (3, −2). a⃗·b⃗ = 2·3 + 3·(−2) = 6−6 = 0 → Perpendikulyar ✓

Nümunə 2: Paralel?

a⃗ = (1, 2), b⃗ = (2, 4). 1·4 − 2·2 = 4−4 = 0 → Paralel ✓ (b⃗ = 2·a⃗)

§9. Vektorun proyeksiyası

b⃗-nin a⃗ üzərindəki proyeksiyası
$$\text{pr}_{\vec{a}}\vec{b} = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}|}$$
Proyeksiya — b⃗-nin a⃗ istiqamətindəki "kölgəsi"
Proyeksiya vektoru
$$\vec{p} = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}|^2} \cdot \vec{a}$$

Nümunə

a⃗ = (3, 0), b⃗ = (2, 4). a⃗·b⃗ = 6. |a⃗| = 3. pr_a(b⃗) = 6/3 = 2

§10. Tətbiqlər: Ortanöqtə və üçbucağın ağırlıq mərkəzi

AB parçasının ortanöqtəsi M
$$M = \left(\frac{x_A + x_B}{2},\; \frac{y_A + y_B}{2}\right)$$
OM⃗ = (OA⃗ + OB⃗) / 2
Üçbucağın ağırlıq mərkəzi G
$$G = \left(\frac{x_A+x_B+x_C}{3},\; \frac{y_A+y_B+y_C}{3}\right)$$
OG⃗ = (OA⃗ + OB⃗ + OC⃗) / 3

Nümunə 1: Ortanöqtə

A(2, 4) və B(8, 0). M = ((2+8)/2, (4+0)/2) = (5, 2)

Nümunə 2: Ağırlıq mərkəzi

A(0, 0), B(6, 0), C(3, 6). G = ((0+6+3)/3, (0+0+6)/3) = (3, 2)
