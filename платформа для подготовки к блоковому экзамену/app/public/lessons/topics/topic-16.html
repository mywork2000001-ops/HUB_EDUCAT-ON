§1. Funksiya anlayışı

Tərif

Əgər X çoxluğunun hər elementinə Y çoxluğundan yalnız bir element uyğun gəlirsə, bu uyğunluğa funksiya deyilir.
Yazılışı: y = f(x)

Əsas terminlər

  • Arqument (x) — müstəqil dəyişən; X çoxluğunun elementi
  • Funksiya qiyməti f(x) — x-ə uyğun y dəyəri
  • Tərif oblastı D(f) — x-in bütün icazə verilən dəyərləri çoxluğu
  • Dəyərlər çoxluğu E(f) — y-in qəbul etdiyi bütün dəyərlər
Funksiya şərti
\(\forall x \in D(f)\;\; \exists!\, y \in E(f): y = f(x)\)
Hər x-ə yalnız bir y uyğun gəlməlidir (unikallıq şərti)

§2. Koordinat müstəvisi və funksiya qrafiki

Funksiya qrafiki

y = f(x) funksiyasının qrafiki — (x; f(x)) koordinatlı bütün nöqtələrin həndəsi yeridir.

Koordinat müstəvisi elementləri

  • Ox — absis oxu (üfüqi), Oy — ordinat oxu (şaquli)
  • Kəsişmə nöqtəsi — başlanğıc nöqtəsi O(0; 0)
  • Koordinatlar: (x; y) — əvvəl üfüqi, sonra şaquli

§3. Xətti funksiya — y = kx + b

Tərif

Xətti funksiya — y = kx + b formasında yazılan funksiyadır (k ≠ 0).
Qrafiki — düz xəttdir.

Əmsalların mənası

  • k — meyl əmsalı (angle coefficient): xəttin maillik bucağını müəyyən edir
  • b — Oy oxunu kəsmə nöqtəsi: y = b nöqtəsindən keçir (x=0)
Standart forma
y = k · x + b
k — meyl əmsalı; b — sərbəst həd; tərif oblastı: \(D(f) = \mathbb{R}\)
SLAYD 4 / 15

§4. Meyl əmsalının rolu (k)

k-nın işarəsinə görə qrafik

  • k > 0 → xətt soldan sağa yuxarı gedir (artan funksiya)
  • k < 0 → xətt soldan sağa aşağı gedir (azalan funksiya)
  • k = 0 → xətt üfüqidir: y = b (sabit funksiya)

|k|-nin qiyməti nə verir?

  • |k| böyük → xətt dik (böyük mail bucağı)
  • |k| kiçik → xətt yatıq (kiçik mail bucağı)
  • k = tan α, burada α — xəttin Ox oxu ilə düzgün istiqamətdə əmələ gətirdiyi bucaq

§5. Xətti funksiyanın qrafiki — cədvəl üsulu

y = f(x) = 2x − 4 funksiyasının qrafikini qurmaq üçün ən azı 2 nöqtə kifayətdir. Ən sadəsi: ox kəsmələri.

Addımlar

  1. x = 0 qoy → y = 2·0 − 4 = −4 → nöqtə A(0; −4)
  2. y = 0 qoy → 0 = 2x − 4 → x = 2 → nöqtə B(2; 0)
  3. A və B nöqtələrini birləşdir
Qiymət cədvəli: y = 2x − 4
x: −1   0   1   2   3
y: −6  −4  −2   0   2

§6. Kvadratik funksiya — y = ax²

Tərif

Kvadratik funksiya — y = ax² + bx + c (a ≠ 0) formasındakı funksiyadır.
Qrafiki — paraboladır.

Aparıcı əmsal a-nın rolu

  • a > 0 → parabola aşağı açıqdır (∪ şəkilli), minimum nöqtəsi var
  • a < 0 → parabola yuxarı açıqdır (∩ şəkilli), maksimum nöqtəsi var
  • |a| böyük → parabola dar; |a| kiçik → parabola geniş

§7. Parabolanın təpəsi və simmetriya oxu

y = ax² + bx + c üçün:

Simmetriya oxu (x₀)
x₀ = −b / (2a)
Parabola bu düz xəttə görə simmetrikdir: x = x₀
Təpə nöqtəsinin koordinatları
T(x₀ ; y₀), y₀ = f(x₀) = c − b² / (4a)
a>0: T minimaldır; a<0: T maksimumdur

Misal: y = x² − 4x + 3

a = 1, b = −4, c = 3

x₀ = −(−4) / (2·1) = 4/2 = 2

y₀ = 2² − 4·2 + 3 = 4 − 8 + 3 = −1

Təpə: T(2; −1), simmetriya oxu: x = 2

§9. Hiperbol — y = k/x

Tərif

Hiperbol — y = k/x (k ≠ 0) funksiyasının qrafikidir.
Tərif oblastı: \(D(f) = \mathbb{R} \setminus \{0\}\), yəni x ≠ 0.

Asimptotlar

  • Üfüqi asimptot: y = 0 (Ox oxu) — qrafikə yaxınlaşır, amma toxunmur
  • Şaquli asimptot: x = 0 (Oy oxu) — qrafikə yaxınlaşır, amma toxunmur
k-nın işarəsi
k > 0: I, III rüblər     k < 0: II, IV rüblər
SLAYD 10 / 15

§10. Hiperbolun xassələri

  • Tərif oblastı: D(f) = (−∞; 0) ∪ (0; +∞)
  • Dəyərlər çoxluğu: E(f) = (−∞; 0) ∪ (0; +∞)
  • Tek funksiya: f(−x) = −f(x) → mərkəz simmetriyası (O(0;0) ilə)
  • k > 0: I, III rüblərdə; azalır hər rübdə ayrılıqda
  • k < 0: II, IV rüblərdə; azalır hər rübdə ayrılıqda
Ümumi formula
y = k / x,   k ≠ 0,   x ≠ 0
y · x = k = const (sabit — hiperbolun tərifi)

Nümunə: y = 12/x

§11. Funksiyanın tək/cüt olması

Cüt funksiya

f(−x) = f(x) bütün x ∈ D(f) üçün keçərlidirsə, funksiya cütdür.
Qrafik Oy oxuna görə simmetrikdir. Misal: y = x², y = |x|

Tək funksiya

f(−x) = −f(x) bütün x ∈ D(f) üçün keçərlidirsə, funksiya təkdir.
Qrafik O(0;0) nöqtəsinə görə simmetrikdir. Misal: y = x³, y = x, y = 1/x

Yoxlama üsulu

  • f(−x) hesabla
  • f(−x) = f(x) → cüt funksiya
  • f(−x) = −f(x) → tək funksiya
  • Nə biri, nə digəri → nə tək, nə cüt

§12. Funksiyanın artması/azalması (monotoniklik)

Artan funksiya

X ⊂ D(f) aralığında: x₁ < x₂ ⇒ f(x₁) < f(x₂) — hər böyük x-ə böyük y uyğundur.

Azalan funksiya

X ⊂ D(f) aralığında: x₁ < x₂ ⇒ f(x₁) > f(x₂) — hər böyük x-ə kiçik y uyğundur.

Xətti funksiyanın monotonikliyi

  • k > 0 → bütün \(\mathbb{R}\)-də artar
  • k < 0 → bütün \(\mathbb{R}\)-də azalar

Kvadratik funksiyanın monotonikliyi (a > 0)

  • (−∞; x₀) aralığında — azalar
  • (x₀; +∞) aralığında — artar
  • x₀ nöqtəsində — minimum (ymin = y₀)

Funksiyanın sıfır nöqtəsi (kökü)

f(x) = 0 tənliyinin hər bir həlli — funksiyanın sıfır nöqtəsidir.
Qrafi üzərindəki Ox oxu ilə kəsişmə nöqtəsidir.

§14. Funksiyanın qrafik üzrə tədqiqi

Qrafik oxunarsa aşağıdakıları müəyyən etmək olar:

  • Tərif oblastı — qrafikin x oxu boyunca uzandığı aralıq
  • Dəyərlər çoxluğu — qrafikin y oxu boyunca uzandığı aralıq
  • Sıfır nöqtələri — qrafikin Ox oxunu kəsdiyi x dəyərləri
  • Müsbət/mənfi aralıqlar — qrafikin Ox-dan yuxarı/aşağı olduğu yerlər
  • Artan/azalan aralıqlar — soldan sağa yuxarı/aşağı gediş
  • Ekstremumlar — lokal minimum və maksimum nöqtələri
İşarə sxemi (y-in müsbətliyi)
x: −∞ ────x₁──── x₂ ──── +∞
y: (+)  (−)   (+)
x₁, x₂ — sıfır nöqtələri; parabola a>0 üçün
