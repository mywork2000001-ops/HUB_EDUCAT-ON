İşlənmiş misallar

Hər misalı özünüz həll etməyə çalışın, sonra həlli açın.

Misal 1Xətti bərabərsizlik — §1
\(3x - 7 > 2\) bərabərsizliyini həll edin.
Həll:
\(3x - 7 > 2\)
\(3x > 2 + 7\)
\(3x > 9\)
\(x > 3\)

Cavab: \(\boxed{x \in (3; +\infty)}\)
Misal 2Kvadrat bərabərsizlik — §3
\(x^2 - 5x + 6 < 0\) bərabərsizliyini həll edin.
Addım 1 — köklər:
\(x^2 - 5x + 6 = 0 \Rightarrow (x-2)(x-3)=0\)
\(x_1 = 2,\quad x_2 = 3\)

Addım 2 — parabolanın yönü: a=1 > 0 → ağız yuxarı
Parabolanın qüsuru 2 ilə 3 arasında < 0-dır.

Cavab: \(\boxed{x \in (2; 3)}\)
Misal 3Kvadrat bərabərsizlik — §3
\(x^2 - 4x + 4 \geq 0\) bərabərsizliyini həll edin.
Addım 1: \(x^2 - 4x + 4 = (x-2)^2 \geq 0\)

Müşahidə: Tam kvadrat həmişə ≥ 0-dır. Yalnız x=2 olduqda = 0 olur.

Cavab: \(\boxed{x \in (-\infty; +\infty)}\) — bütün real ədədlər.
Misal 4Rasional bərabərsizlik — §4
\(\dfrac{x-3}{x+1} > 0\) bərabərsizliyini həll edin.
Kritik nöqtələr: x = 3 (sıfır), x = −1 (mənfi tərəf = 0 olur → məxrəc sıfır)

İnterval üsulu:
say oxu üzərində: (−∞; −1), (−1; 3), (3; +∞)
• x = −2: \(\frac{-5}{-1} = 5 > 0\) ✓
• x = 0: \(\frac{-3}{1} = -3 < 0\) ✗
• x = 4: \(\frac{1}{5} > 0\) ✓

Cavab: \(\boxed{x \in (-\infty; -1) \cup (3; +\infty)}\)
Misal 5Modullu bərabərsizlik — §5
\(|2x - 1| < 5\) bərabərsizliyini həll edin.
Qayda: \(|f(x)| < a \Leftrightarrow -a < f(x) < a\)

\(-5 < 2x - 1 < 5\)
\(-5 + 1 < 2x < 5 + 1\)
\(-4 < 2x < 6\)
\(-2 < x < 3\)

Cavab: \(\boxed{x \in (-2; 3)}\)
Misal 6Bərabərsizliklər sistemi — §6
\(\begin{cases} 2x + 3 > 7 \\ x - 4 \leq 2 \end{cases}\) sistemini həll edin.
Birinci: \(2x + 3 > 7 \Rightarrow 2x > 4 \Rightarrow x > 2\)
İkinci: \(x - 4 \leq 2 \Rightarrow x \leq 6\)

Kəsişmə: \(2 < x \leq 6\)

Cavab: \(\boxed{x \in (2; 6]}\)
