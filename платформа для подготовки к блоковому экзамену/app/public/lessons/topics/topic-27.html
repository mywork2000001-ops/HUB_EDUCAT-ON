İntegral (Hədd Hesabı)

Qeyri-müəyyən və müəyyən inteqral — antitöremə, Newton-Leybnits düsturu, sahə hesabı

Slayd 1 / ?
SLIDE 01

Antitöremə (İbtidai Funksiya)

Tərif

$F(x)$ funksiyası $f(x)$-in antitöreməsidir, əgər $F'(x) = f(x)$ bütün $x$ üçün doğrudursa.

Ümumi antitöremə
$$F(x) + C \quad (C = \text{sabit})$$
$C$ ixtiyari sabitdir. $F(x)+C$-nin törəməsi $f(x)$-dir.

Nümunə

$f(x) = 2x$ üçün antitöremə: $F(x) = x^2 + C$ (çünki $(x^2+C)' = 2x$)

  • Antitöremə sonsuz sayda ola bilər ($C$ fərqli olduqda)
  • Başlanğıc şərt verilsə, konkret $C$ tapılır
SLIDE 02

İnteqral Cədvəli

$f(x)$$\int f(x)\,dx$
$x^n$ ($n\neq-1$)$\dfrac{x^{n+1}}{n+1} + C$
$\dfrac{1}{x}$$\ln|x| + C$
$e^x$$e^x + C$
$a^x$$\dfrac{a^x}{\ln a} + C$
$\sin x$$-\cos x + C$
$\cos x$$\sin x + C$
$\dfrac{1}{\cos^2 x}$$\tan x + C$
$\dfrac{1}{\sin^2 x}$$-\cot x + C$
SLIDE 03

İnteqralın Xassələri

  • $\displaystyle\int k\,f(x)\,dx = k\int f(x)\,dx$ — sabitlə vurma
  • $\displaystyle\int [f(x)\pm g(x)]\,dx = \int f(x)\,dx \pm \int g(x)\,dx$ — cəm/fərq
  • $\left(\displaystyle\int f(x)\,dx\right)' = f(x)$ — inteqral törəməsinin inversidir
  • $\displaystyle\int f'(x)\,dx = f(x) + C$ — törəmənin inteqralı

Nümunə

$\displaystyle\int (3x^2 - 2x + 5)\,dx = x^3 - x^2 + 5x + C$

SLIDE 04

Əvəzetmə Üsulu (u-Əvəzetmə)

Əvəzetmə düsturu
$$\int f(g(x))\cdot g'(x)\,dx = \int f(u)\,du \Big|_{u=g(x)}$$

Alqoritm

  1. Daxili funksiyanı $u = g(x)$ kimi seç
  2. $du = g'(x)\,dx$ tap
  3. İnteqralı $u$ ilə yaz
  4. İnteqralı hesabla, sonra $u = g(x)$ geri qoy

Nümunə

$\displaystyle\int 2x e^{x^2}\,dx$: $u=x^2$, $du=2x\,dx$ → $\displaystyle\int e^u\,du = e^u+C = $ $e^{x^2}+C$

SLIDE 05

Hissə-Hissə İnteqrasiya

Hissə-hissə düstur
$$\int u\,dv = uv - \int v\,du$$

Seçim qaydası (LIATE)

$u$ üçün: Logarifm → Invers trig → Algebraik → Triqonometrik → Eksponensial

Nümunə

$\displaystyle\int x e^x\,dx$: $u=x$, $dv=e^x dx$ → $du=dx$, $v=e^x$

$= xe^x - \displaystyle\int e^x\,dx = xe^x - e^x + C = $ $(x-1)e^x + C$

SLIDE 06

Müəyyən İnteqral və Newton-Leybnits Düsturu

Newton-Leybnits düsturu
$$\int_a^b f(x)\,dx = F(b) - F(a) = \bigl[F(x)\bigr]_a^b$$
$F$ — $f$-in ixtiyari antitöreməsi

Nümunə

$\displaystyle\int_1^3 2x\,dx = [x^2]_1^3 = 9 - 1 = $ 8

Xassələr

  • $\displaystyle\int_a^a f(x)\,dx = 0$
  • $\displaystyle\int_a^b f(x)\,dx = -\int_b^a f(x)\,dx$
  • $\displaystyle\int_a^c f\,dx = \int_a^b f\,dx + \int_b^c f\,dx$
SLIDE 07

Əyri Altındakı Sahə

Sahə düsturu
$$S = \int_a^b |f(x)|\,dx$$
$f(x)\geq 0$ olduqda: $S = \int_a^b f(x)\,dx$

Mənfi sahə

Funksiya $x$-oxunun altında ($f(x)<0$) olduqda, sahə tapılarkən mütləq dəyər alınır. İnteqral mənfi ola bilər, amma sahə həmişə müsbətdir.

Nümunə

$y=x^2-1$, $[0,2]$ arasındakı sahə:

$\displaystyle\int_0^1|x^2-1|\,dx + \int_1^2(x^2-1)\,dx = \frac{2}{3}+\frac{4}{3} = 2$

SLIDE 08

İki Əyri Arasındakı Sahə

İki əyri arasındakı sahə
$$S = \int_a^b |f(x) - g(x)|\,dx$$
$f(x)\geq g(x)$ olduqda mütləq dəyər lazım deyil

Alqoritm

  1. $f(x)=g(x)$ tənliyini həll et → kəsişmə nöqtələri ($a$, $b$)
  2. Hansı funksiyanın üstdə olduğunu müəyyən et
  3. $S = \displaystyle\int_a^b [f(x)-g(x)]\,dx$

Nümunə

$f(x)=x$, $g(x)=x^2$, kəsişmə: $x=0,1$: $S=\displaystyle\int_0^1(x-x^2)\,dx=\left[\frac{x^2}{2}-\frac{x^3}{3}\right]_0^1=\frac{1}{6}$

SLIDE 09

Qeyri-Xas İnteqrallar

Tərif

İnteqrasiya intervalı sonsuz olduqda və ya integrand hərhansi nöqtədə ∞ olduqda qeyri-xas inteqraldan söhbət gedir.

Sonsuz həddli qeyri-xas inteqral
$$\int_a^{+\infty} f(x)\,dx = \lim_{t\to+\infty} \int_a^t f(x)\,dx$$

Nümunə

$\displaystyle\int_1^{+\infty}\frac{1}{x^2}\,dx = \lim_{t\to\infty}\left[-\frac{1}{x}\right]_1^t = \lim_{t\to\infty}\left(-\frac{1}{t}+1\right) = $ 1 (yığılır)

SLIDE 10

Disk Üsulu — Fırlanma Həcmi

x-oxu ətrafında fırlanma həcmi
$$V = \pi \int_a^b [f(x)]^2\,dx$$

Nümunə

$y=\sqrt{x}$ qrafiki, $[0,4]$ arası, $x$-oxu ətrafında fırlanır:

$V = \pi\displaystyle\int_0^4 x\,dx = \pi\cdot\left[\frac{x^2}{2}\right]_0^4 = \pi\cdot8 = $ $8\pi$

Üzük üsulu

Boş silindir üçün: $V=\pi\displaystyle\int_a^b([f(x)]^2-[g(x)]^2)\,dx$

SLIDE 11

Əyri Altındakı Sahə — SVG

$y=x^2$, $[0,2]$ intervalındakı sahə: $\displaystyle\int_0^2 x^2\,dx = \left[\frac{x^3}{3}\right]_0^2 = \frac{8}{3} \approx $ 2.67
