İşlənmiş misallar

Hər misalı özünüz həll etməyə çalışın, sonra həlli açın.

Misal 1Arifmetik silsilə — n-ci həd
Arifmetik silsilənin 1-ci hədi \(a_1 = 3\), fərqi \(d = 5\). 10-cu hədi tapın.
Düstur: \(a_n = a_1 + (n-1)d\)

\(a_{10} = 3 + (10-1) \cdot 5 = 3 + 45 = 48\)

Cavab: \(\boxed{a_{10} = 48}\)
Misal 2Arifmetik silsilə — cəm
Arifmetik silsilənin \(a_1 = 2\), \(d = 3\) olduqda ilk 15 hədinin cəmini tapın.
Düstur: \(S_n = \dfrac{n}{2}(2a_1 + (n-1)d)\)

\(S_{15} = \dfrac{15}{2}(2\cdot2 + 14\cdot3) = \dfrac{15}{2}(4+42) = \dfrac{15\cdot46}{2} = 345\)

Cavab: \(\boxed{S_{15} = 345}\)
Misal 3Həndəsi silsilə — n-ci həd
Həndəsi silsilənin \(b_1 = 4\), \(q = 3\) olduqda 5-ci hədi tapın.
Düstur: \(b_n = b_1 \cdot q^{n-1}\)

\(b_5 = 4 \cdot 3^{4} = 4 \cdot 81 = 324\)

Cavab: \(\boxed{b_5 = 324}\)
Misal 4Həndəsi silsilə — cəm
Həndəsi silsilənin \(b_1 = 2\), \(q = 3\) olduqda ilk 5 hədinin cəmini tapın.
Düstur: \(S_n = b_1 \cdot \dfrac{q^n - 1}{q - 1}\) (\(q \neq 1\))

\(S_5 = 2 \cdot \dfrac{3^5-1}{3-1} = 2 \cdot \dfrac{243-1}{2} = 2 \cdot 121 = 242\)

Cavab: \(\boxed{S_5 = 242}\)
Misal 5Sonsuz həndəsi silsilə
\(b_1 = 12\), \(q = \dfrac{1}{3}\) olan sonsuz həndəsi silsilənin cəmini tapın.
Şərt: \(|q| < 1\) olduqda sonsuz cəm mövcuddur.
\(|q| = \frac{1}{3} < 1\) ✓

Düstur: \(S_\infty = \dfrac{b_1}{1-q}\)

\(S_\infty = \dfrac{12}{1 - \frac{1}{3}} = \dfrac{12}{\frac{2}{3}} = 12 \cdot \dfrac{3}{2} = 18\)

Cavab: \(\boxed{S_\infty = 18}\)
Misal 6DİM — bilinməyəni tap
Arifmetik silsilənin 3-cü hədi 11, 7-ci hədi 23-dür. Fərqi və 1-ci hədi tapın.
Sistem:
\(a_3 = a_1 + 2d = 11\)
\(a_7 = a_1 + 6d = 23\)

Çıxarırıq: \(4d = 12 \Rightarrow d = 3\)
\(a_1 = 11 - 2 \cdot 3 = 5\)

Cavab: \(\boxed{d = 3, \; a_1 = 5}\)
