Vieta Teoreminin İsbatı

İkinci dərəcəli tənliyin köklərini əmsallarla əlaqələndirən əsas teorem

Teorem (François Viète, 1591)
Əgər x₁ və x₂ — ax² + bx + c = 0 tənliyinin kökləri olsarsa (a ≠ 0):
Köklər cəmi
x₁ + x₂ = −b/a
Köklər hasili
x₁ · x₂ = c/a
1

Çarpanlara ayırma

Əgər x₁ və x₂ tənliyin kökləri olsarsa, onda polinom bu köklər vasitəsilə çarpanlara ayrıla bilər:

ax² + bx + c = a(x − x₁)(x − x₂)

Bu əlaqə cəbrin əsas teoremindən irəli gəlir: hər köklü polinom bu şəkildə yazıla bilər.

2

Sağ tərəfi açırıq

Mötərizəni açırıq:

a(x − x₁)(x − x₂)
= a · [x² − x₁x − x₂x + x₁x₂]
= a · [x² − (x₁ + x₂)x + x₁x₂]
= ax² − a(x₁+x₂)·x + a·x₁x₂
3

Əmsalları müqayisə edirik

Sol tərəf: ax² + bx + c
Sağ tərəf: ax² − a(x₁+x₂)·x + a·x₁x₂

x¹ əmsallarını bərabər qoyuruq:

b = −a(x₁ + x₂)
⟹ x₁ + x₂ = −b/a ✓

Sərbəst həddləri bərabər qoyuruq:

c = a · x₁x₂
⟹ x₁ · x₂ = c/a ✓
4

Nümunə ilə yoxlama

Nümunə: x² − 5x + 6 = 0 (a=1, b=−5, c=6)

Vieta: x₁ + x₂ = 5/1 = 5; x₁ · x₂ = 6/1 = 6

Köklər tapaq: x² − 5x + 6 = (x−2)(x−3) = 0 → x₁=2, x₂=3

Yoxlama: 2 + 3 = 5 ✓ (cəm düzgün)
Yoxlama: 2 × 3 = 6 ✓ (hasil düzgün)

Praktik istifadə: Vieta teoremi köklər tapılmadan tənlik haqqında məlumat verir. Əgər x₁+x₂ = 7 və x₁·x₂ = 12 bilinərsə, köklər 3 və 4-dür — həll etmədən!
