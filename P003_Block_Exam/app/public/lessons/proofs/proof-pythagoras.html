Pifaqor Teoreminin İsbatı

Həndəsənin ən məşhur teoremlərindən birinin vizual isbatı

Teorem
Düzbucaqlı üçbucaqda katetlərinin kvadratlarının cəmi hipotenuzun kvadratına bərabərdir.
a² + b² = c²
burada a, b — katetlər; c — hipotenuz (düzbucağın qarşısındakı tərəf)
1

Konstruksiya

a kateti, b kateti və c hipotenuzlu 4 eyni düzbucaqlı üçbucağı götürək. Onları (a+b) tərəfli böyük kvadratın içinə yerləşdirək:

2

Böyük kvadratın sahəsi iki üsulla

Böyük kvadrat tərəfi (a+b)-dir. Sahəsini iki müxtəlif üsulla ifadə edək:

Üsul 1: S_böyük = (a + b)² = a² + 2ab + b²
Üsul 2: S_böyük = 4 × (üçbucaq sahəsi) + c²
= 4 × (ab/2) + c² = 2ab + c²
3

Bərabərlik quruluşu

İki üsul eyni sahəni verir, buna görə onları bərabər qoyuruq:

a² + 2ab + b² = 2ab + c²

Hər iki tərəfdən 2ab çıxırıq:

a² + b² = c² ✓
4

Nəticə

Beləliklə, düzbucaqlı üçbucaqda katetlərin kvadratlarının cəmi hipotenuzun kvadratına bərabərdir. İsbat tamamdır.

Tətbiq nümunəsi: Əgər a = 3, b = 4 olarsa: c² = 9 + 16 = 25, deməli c = 5 sm. Bu, ən məşhur Pifaqor üçlüyüdür: 3–4–5.
