📚Десятичные дроби

Десятичная дробь — это дробь со знаменателем 10, 100, 1000 и т.д., записанная без знаменателя. Целая часть отделяется запятой.

3,78 = 3 + 7/10 + 8/100 = 3 + 0,7 + 0,08

Разряды после запятой: первая — десятые, вторая — сотые, третья — тысячные.

Нули в конце десятичной дроби не меняют её значения: 0,50 = 0,5 = 0,500.
⚖️Сравнение

Сравнивайте по разрядам слева направо. Если разрядов не хватает — допишите нули.

0,3 > 0,299, потому что 3 десятых больше 2 десятых.
0,182 > 0,1078, потому что 8 сотых больше 0 сотых.
Не путайте количество цифр с величиной. 0,028 (3 знака) < 0,19 (2 знака), так как 0,028 = 28 тысячных, а 0,19 = 190 тысячных.
➕➖Сложение и вычитание
  1. Запишите числа друг под другом так, чтобы запятые были на одной вертикали.
  2. Допишите нули справа, чтобы разрядность совпадала.
  3. Складывайте/вычитайте как целые числа.
  4. В ответе поставьте запятую под запятыми.
3,783
+ 2,200
-------
5,983
Если складываете 3,783 + 2,2, допишите нули: 2,2 = 2,200.
🔀Перевод из обыкновенных

Если знаменатель — 10, 100, 1000 — просто запишите числитель, отделив нужное количество цифр запятой справа.

710 = 0,7     34 = 0,75 (3÷4)     18 = 0,125 (1÷8)

Сокращение дробей перед переводом: делите числитель и знаменатель на их НОД.

615 = 25 = 0,4     2135 = 35 = 0,6     4256 = 34 = 0,75
