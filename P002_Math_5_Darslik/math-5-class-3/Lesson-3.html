Правило сравнения

Сравнение десятичных дробей по разрядам: целая часть, десятые, сотые, тысячные.

Числовая прямая

Расположение дробей на прямой. Меньшее число левее, большее — правее.

Знаки >, <, =

Постановка правильного знака между двумя десятичными дробями.

Задачи с пропусками

Восстановление цифры в записи дроби, чтобы неравенство было верным.

📚 Основное правило

Чтобы сравнить две десятичные дроби, сравнивают их по разрядам слева направо:

  1. Сначала сравнивают целые части.
  2. Если целые части равны — сравнивают десятые.
  3. Если десятые равны — сравнивают сотые.
  4. И так далее, пока не найдут различающийся разряд.
3,42 < 3,45
Целые равны (3=3), десятые равны (4=4), сотые: 2 < 5
Можно дописать нули справа, чтобы разрядов было поровну: 3,5 = 3,50 = 3,500.
