➕ Переместительное свойство сложения

От перестановки слагаемых сумма не изменяется.

a + b = b + a

Пример: 347 + 253 = 253 + 347 = 600

578 + 422
= 1 000
422 + 578
= 1 000
Используйте это свойство для выбора удобного порядка: сначала складывайте числа, дающие «круглую» сумму.

🔗 Сочетательное свойство сложения

При сложении нескольких чисел группировка слагаемых не меняет результат.

(a + b) + c = a + (b + c)

Пример: (148 + 52) + 300 = 200 + 300 = 500

Или: 148 + (52 + 300) = 148 + 352 = 500

Группируйте числа так, чтобы промежуточные суммы были «круглыми»: 148 + 52 = 200 — удобно для устного счёта!

➖ Вычитание — обратная операция

Вычитание — нахождение неизвестного слагаемого. Если a + b = c, то c − b = a.

a − b = c  ⟺  a = b + c

Компоненты вычитания:

a
уменьшаемое
b
вычитаемое
c
разность
Вычитание не обладает переместительным свойством: a − b ≠ b − a (если a ≠ b).

Пример: 8 035 − 2 478

Когда цифра уменьшаемого меньше цифры вычитаемого, занимаем единицу из следующего разряда, прибавляем 10 к текущему.

Проверка: 5 557 + 2 478 = 8 035 ✓ — разность плюс вычитаемое равна уменьшаемому.

✅ Проверка результата

Проверка сложения: сумма − слагаемое = другое слагаемое.

3 478 + 2 656 = 6 134
Проверка: 6 134 − 2 656 = 3 478 ✓

Проверка вычитания: разность + вычитаемое = уменьшаемое.

8 035 − 2 478 = 5 557
Проверка: 5 557 + 2 478 = 8 035 ✓

📝 Применение в задачах

Большинство текстовых задач сводится к сложению или вычитанию.

«На сколько больше/меньше?»

→ Вычитание: большее − меньшее

«Сколько всего/вместе?»

→ Сложение: часть₁ + часть₂

«Сколько осталось?»

→ Вычитание: было − взяли

«Найдите неизвестное слагаемое»

→ Вычитание: сумма − известное слагаемое

📋 Сложение многозначных чисел (столбиком)

Алгоритм сложения в столбик:

1. Записываем числа одно под другим, выравнивая по разрядам.
2. Складываем справа налево (единицы, десятки, ...).
3. Если сумма ≥ 10 — пишем единицы, переносим 1 в следующий разряд.
3 745 + 2 867 = 6 612 (столбиком с переносами)
Всегда выравнивай цифры по правому краю — единицы под единицами!

📋 Вычитание многозначных чисел (столбиком)

Алгоритм вычитания в столбик:

1. Записываем числа в столбик (большее сверху).
2. Вычитаем справа налево.
3. Если цифра уменьшаемого меньше — занимаем 1 из следующего разряда.
7 003 − 1 458 = 5 545 (с заимствованиями)
Нули в числе уменьшаемого требуют «цепочки заимствований».

🔄 Взаимосвязь сложения и вычитания

Сложение и вычитание — взаимно обратные операции.

a + b = c  ⟺  c − b = a  ⟺  c − a = b

Это свойство используется для проверки: если a + b = c, то c − b должно дать a.

Всегда проверяй сложение вычитанием и наоборот — так ловят ошибки!

🧮 Уравнения на сложение и вычитание

Уравнения вида x + a = b или x − a = b решаются через взаимосвязь операций:

x + a = b  →  x = b − a
x − a = b  →  x = b + a
a − x = b  →  x = a − b

После нахождения x обязательна проверка подстановкой.

Неизвестное слагаемое = сумма − другое слагаемое. Это главное правило!

✅ Практика: задачи с неизвестным слагаемым

Решаем задачи на сложение и вычитание:

Задача 1. x + 358 = 742 → x = 742 − 358 = 384
Задача 2. 1 000 − x = 437 → x = 1 000 − 437 = 563
Задача 3. В библиотеке было 3 420 книг. Выдали 847. Осталось: 3 420 − 847 = 2 573
Проверяй: 384 + 358 = 742 ✓   563 + 437 = 1 000 ✓   2 573 + 847 = 3 420 ✓
🪜Глава 2. Пошаговые алгоритмы

Пятиступенчатый алгоритм для любой задачи Урока 5.

1

Определите операцию

Прочитайте задачу. Ключевые слова «всего», «вместе» → сложение. «Осталось», «на сколько больше» → вычитание.

2

Сгруппируйте удобно

Используйте переместительное и сочетательное свойства — перестройте слагаемые так, чтобы промежуточные суммы были «круглыми» (кратными 10, 100, 1000).

3

Выполните вычисление в столбик

Запишите числа одно под другим, выравнивая по разрядам (единицы под единицами). Выполните действие справа налево, учитывая перенос (при сложении) или заимствование (при вычитании).

4

Проверьте результат

Сложение: вычтите одно слагаемое из суммы — должно получиться другое слагаемое. Вычитание: прибавьте вычитаемое к разности — должно получиться уменьшаемое.

5

Запишите ответ с единицами

Убедитесь, что ответ имеет смысл (нельзя «получить отрицательное количество» при работе с натуральными числами). Укажите единицу измерения, если она задана.

№ 1Удобное сложение
Вычислите удобным способом: 29 + 23 + 107 + 41
1Ищем пары, дающие «круглую» сумму: 29 + 41 = 70, 23 + 107 = 130.
270 + 130 = 200.
✓Ответ: 200
№ 2Сложение в столбик
Найдите сумму: 2 862 + 6 137
1Единицы: 2 + 7 = 9 → пишем 9, переноса нет.
2Десятки: 6 + 3 = 9 → пишем 9.
3Сотни: 8 + 1 = 9 → пишем 9.
4Тысячи: 2 + 6 = 8 → пишем 8.
✓Ответ: 8 999
№ 3Вычитание в столбик
Найдите разность: 96 586 − 80 235
1Единицы: 6 − 5 = 1 → пишем 1.
2Десятки: 8 − 3 = 5 → пишем 5.
3Сотни: 5 − 2 = 3 → пишем 3.
4Тысячи: 6 − 0 = 6 → пишем 6.
5Десятки тысяч: 9 − 8 = 1 → пишем 1.
✓Ответ: 16 351
№ 4Метод компенсации
Вычислите: 4 960 − 1 999
1Округляем вычитаемое: 1 999 ≈ 2 000.
24 960 − 2 000 = 2 960.
3Компенсируем: прибавляем 1 (т.к. вычли на 1 больше): 2 960 + 1 = 2 961.
✓Ответ: 2 961
№ 5Нахождение неизвестного слагаемого
Найдите неизвестное: □ + 253 = 600
1Неизвестное слагаемое = сумма − известное слагаемое.
2□ = 600 − 253 = 347.
3Проверка: 347 + 253 = 600 ✓.
✓Ответ: 347
№ 6Задача
В библиотеке было 2 372 книги. Привезли ещё 3 514. Сколько книг стало всего? На сколько стало больше?
1Всего книг: 2 372 + 3 514 = 5 886.
2На сколько больше: 3 514 − 2 372 = 1 142.
✓Ответ: стало 5 886 книг, больше на 1 142 книги.
