Айнур устно определила, что ответ на пример, записанный на доске, равен 500.

1 + 2 + 3 + 4 + 5 + 95 + 96 + 97 + 98 + 99 = ?

Как она это определила?

Она заметила удобные пары:

1 + 99 = 100 2 + 98 = 100 3 + 97 = 100 4 + 96 = 100 5 + 95 = 100

Модуль 1

Свойства сложения

Сумму и разность можно найти удобным способом, используя свойства сложения.

1. Свойство нуля

2 + 0 = 0 + 2 = 2

Если к числу прибавить нуль, получится само число.

a + 0 = 0 + a = a

2. Переместительное свойство

2 + 3 = 3 + 2

От перестановки мест слагаемых сумма не меняется.

a + b = b + a

Благодаря этому свойству можно менять порядок слагаемых, чтобы сложение было удобнее.

3. Сочетательное свойство

2 + 3 + 4 = (2 + 3) + 4 = 2 + (3 + 4)

При сложении трёх слагаемых два рядом стоящих слагаемых можно заменить их суммой.

a + b + c = (a + b) + c = a + (b + c)

Это свойство позволяет группировать слагаемые в удобные пары.

Вывод

Благодаря этим трём свойствам мы можем:

  • Менять порядок слагаемых (переместительное)
  • Группировать их в удобные пары (сочетательное)
  • Не бояться нуля (свойство нуля)
Модуль 2

Удобный способ сложения

Сложите удобным способом, используя свойства сложения.

Образец

29 + 23 + 107 + 41

Решение:

1
Переставим и сгруппируем удобные пары: (29 + 41) + (23 + 107)
2
Считаем каждую пару: 29 + 41 = 70 и 23 + 107 = 130
3
Складываем результаты: 70 + 130 = 200

Объяснение: Переместительное и сочетательное свойства сложения позволяют складывать числа в удобной последовательности.

Задание 1. Решения

а) 33 + 15 + 17

= (33 + 17) + 15 = 50 + 15 = 65

б) 104 + 50 + 16 + 150

= (104 + 16) + (50 + 150) = 120 + 200 = 320

в) 120 + 240 + 360 + 180

= (120 + 180) + (240 + 360) = 300 + 600 = 900

г) 212 + 546 + 188 + 154

= (212 + 188) + (546 + 154) = 400 + 700 = 1100

Задание 2. Впишите подходящие числа

а) 400 + 0 = 400 (свойство нуля)

б) 33 + 55 = 55 + 33 (переместительное)

в) (3 + 4) + 5 = 3 + (4 + 5) (сочетательное)

г) 9 − (6 − 2) = (9 − 6) + 2 (вычитание разности)

а) a + 7 = 7 + a

б) (b + 10) + 90 = b + 100

Задание 4. Найдите значения, если a + b = 7

а) (a + 3) + b = (a + b) + 3 = 7 + 3 = 10

б) b + (9 + a) = (a + b) + 9 = 7 + 9 = 16

в) a + (b + 2) = (a + b) + 2 = 7 + 2 = 9

г) (a + 5) + (b + 11) = (a + b) + 16 = 7 + 16 = 23

Задание 5. Верны ли утверждения?

а) «Сумма двух любых натуральных чисел больше любого из слагаемых»

Верно. Если оба числа натуральные (начинаются с 1), то сумма всегда больше каждого из слагаемых. Например: 3 + 5 = 8 > 3 и 8 > 5.

б) «Разность двух натуральных чисел всегда больше вычитаемого»

Неверно. Контрпример: 10 − 8 = 2, а 2 < 8 (разность меньше вычитаемого).

Разные методы сложения и вычитания

Задание 6. Развёрнутая форма

Образец

2 372 + 3 514

Решение:

2 372 = 2 000 + 300 + 70 + 2

3 514 = 3 000 + 500 + 10 + 4

Складываем по разрядам:

(2 000 + 3 000) + (300 + 500) + (70 + 10) + (2 + 4) = 5 000 + 800 + 80 + 6 = 5 886

Объяснение: Числа записываются в развёрнутой форме. Соответствующие значения разрядов складываются по переместительному и сочетательному свойствам. Ответы складываются.

а) 2 862 + 6 137 = 8 999

= (2000+6000) + (800+100) + (60+30) + (2+7) = 8000+900+90+9

б) 96 586 − 80 235 = 16 351

= (90000-80000) + (6000-0) + (500-200) + (80-30) + (6-5)

в) 52 578 225 − 41 466 124 = 11 112 101

г) 208 996 − 105 = 208 891

д) 280 053 358 + 44 520 = 280 097 878

е) 485 288 511 + 304 610 482 = 789 898 993

Задание 8. Вычитание по частям (развёрнутая форма вычитаемого)

Образец

652 745 − 550 200

Решение:

652 745 − 550 200 = 652 745 − 500 000 − 50 000 − 200

= 152 745 − 50 000 − 200 = 102 745 − 200 = 102 545

Объяснение: Разрядные значения цифр вычитаемого по частям вычитаются из уменьшаемого (500 000, 50 000, 200).

а) 627 211 − 103 100 = 524 111

= 627211 − 100000 − 3000 − 100 = 527211 − 3000 − 100 = 524211 − 100

б) 2 537 240 + 360 150 = 2 897 390

в) 368 580 866 − 200 380 600 = 168 200 266

Задание 9. Метод компенсации

Образец

365 000 + 197 000

Решение:

Чтобы дополнить второе слагаемое до 200 000, из первого слагаемого вычитается 3 000, а ко второму прибавляется 3 000.

365 000 + 197 000 = (365 000 − 3 000) + (197 000 + 3 000) = 362 000 + 200 000 = 562 000

Объяснение: Сумма не изменяется, если одно слагаемое уменьшить, а другое увеличить на одно и то же число.

а) 2 003 + 1 216

= (2003−3) + (1216+3) = 2000 + 1219 = 3 219

б) 1 280 + 2 030

= (1280+30) + (2030−30) = 1310 + 2000 = 3 310 (или наоборот)

в) 4 960 − 1 999

= 4960 − 2000 + 1 = 2960 + 1 = 2 961

г) 596 000 − 302 000

= (596000+2000) − (302000+2000)? Нет, проще: = 596000 − 300000 − 2000 = 296000 − 2000 = 294 000

Задача 14. Люди, потребляющие меньше суточной нормы пищи (2020 год)

Дано:

  • Африка: 282 000 000 человек
  • Азия: 418 000 000 человек
  • Северная и Южная Америка: 60 000 000 человек

Вопрос: Каково общее число людей?

Решение

282 000 000 + 418 000 000 + 60 000 000

Удобно сначала сложить 282 и 418:

282 + 418 = 700, значит 282 000 000 + 418 000 000 = 700 000 000

Затем: 700 000 000 + 60 000 000 = 760 000 000

Ответ: 760 000 000 человек.
