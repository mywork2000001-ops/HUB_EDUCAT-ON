Свойства умножения

Изучим 6 основных свойств, которые сделают вычисления проще и быстрее.

1. Умножение на ноль

При умножении любого числа на ноль получается ноль.

a · 0 = 0 · a = 0

Пример: 3 · 0 = 0

2. Умножение на единицу

При умножении числа на единицу получается само число.

a · 1 = 1 · a = a

Пример: 5 · 1 = 5

3. Переместительное свойство

От перестановки мест множителей произведение не меняется.

a · b = b · a

Пример: 3 · 5 = 5 · 3 = 15

4. Сочетательное свойство

Два рядом стоящих множителя можно заменить их произведением.

a · b · c = (a · b) · c = a · (b · c)

Пример: 2 · 3 · 5 = (2·3)·5 = 2·(3·5) = 30

5. Распределительное свойство относительно сложения

Произведение суммы на число равно сумме произведений каждого слагаемого на это число.

a · (b + c) = a · b + a · c

Пример: 5 · (2 + 4) = 5·2 + 5·4 = 10 + 20 = 30

6. Распределительное свойство относительно вычитания

Произведение разности на число равно разности произведений уменьшаемого и вычитаемого на это число.

a · (b − c) = a · b − a · c

Пример: 7 · (10 − 3) = 7·10 − 7·3 = 70 − 21 = 49

Свойства деления

Правила деления, которые обязательно нужно запомнить. Особое внимание — делению на ноль!

1. Деление нуля

При делении нуля на натуральное число получается ноль.

0 : a = 0  (a ≠ 0)

Пример: 0 : 3 = 0

2. Деление на ноль

Число на ноль делить нельзя!

a : 0 — не определено

Пример: 3 : 0 — нельзя!

3. Деление на единицу

При делении числа на 1 получается само число.

a : 1 = a

Пример: 4 : 1 = 4

4. Деление суммы на число

Чтобы разделить сумму на число, можно разделить на это число каждое слагаемое и полученные частные сложить.

(a + b) : c = a : c + b : c  (c ≠ 0)

Пример: (20 + 30) : 5 = 20:5 + 30:5 = 4 + 6 = 10

5. Деление разности на число

Чтобы разделить разность на число, можно разделить на это число уменьшаемое и вычитаемое и найти их разность.

(a − b) : c = a : c − b : c  (c ≠ 0)

Пример: (60 − 18) : 6 = 60:6 − 18:6 = 10 − 3 = 7

Подумай!

Как объяснить утверждение «число на 0 делить нельзя!», пользуясь связью умножения и деления?

Деление — это действие, обратное умножению. Если a : 0 = b, то должно быть верно b · 0 = a. Но мы знаем, что любое число · 0 = 0. Значит, a должно быть равно 0. Но даже если a = 0, то b может быть любым числом (5·0=0, 7·0=0...), и ответ неоднозначен. Поэтому деление на ноль не определяется.

Вычислительные приёмы

Приближённые вычисления и удобные способы устного счёта.

Приближённое умножение (Задание №12)

Множители заменяются близкими и удобными для вычисления числами.

Образец

636 · 291 ≈ ?

636 ≈ 600,    291 ≈ 300
636 · 291 ≈ 600 · 300 = 180 000
(Точный ответ: 185 076 — близко к оценке)
11 · 98 ≈ ?
11 ≈ 10, 98 ≈ 100 → 10·100 = 1000
Точно: 1078
788 · 103 ≈ ?
788 ≈ 800, 103 ≈ 100 → 800·100 = 80 000
Точно: 81 164
Приближённое деление (Задание №13)

Делимое и делитель заменяются близкими и удобными числами.

Образец

304 : 16 ≈ ?

304 ≈ 300,    16 ≈ 15
304 : 16 ≈ 300 : 15 = 20
(Точный ответ: 19 — очень близко!)
Проверка правдоподобия (Задание №14)

Проверьте, правдоподобен ли ответ приближёнными вычислениями.

Образец

103 · 49 = 3 027 ?

103 ≈ 100,    49 ≈ 50
100 · 50 = 5 000
Ответ 3 027 неправдоподобен, т.к. разность между 5 000 и 3 027 слишком большая.
(Точный ответ: 5 047)
