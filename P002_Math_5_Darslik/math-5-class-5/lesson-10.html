🔗 Зависимость между величинами

Если от изменения одной величины меняется другая, они зависимы. Первая называется независимой (x), вторая — зависимой (y).

Количество маек (x) → Стоимость (y)
1 маяка = 10 ₼ → y = 10·x
Независимую переменную обычно обозначают x, зависимую — y.

⚙️ Вычислительные машины

Машина преобразует входное число x в выходное y по заданному правилу.

x → +3 → y   →   формула: y = x + 3
x → ×3 → y   →   формула: y = 3x
Если x = 4, то для y = x + 3 получаем y = 7. Для y = 3x получаем y = 12.

📋 Таблицы зависимостей

Таблица показывает, какое значение y соответствует каждому x.

y = 2x
x12345
y246810
Каждой x ставится в соответствие одна y. Это и есть функция.

📐 Формулы из геометрии

Формулы — это тоже зависимости между величинами.

Периметр квадрата: P = 4a
Площадь прямоугольника: S = a · b
Путь: S = v · t (скорость × время)
Если сторона квадрата a = 5, то P = 4·5 = 20.

🔍 Нахождение формулы по таблице

Посмотрите на закономерность между x и y, чтобы найти формулу.

x1234
y3579
Шаг: y растёт на 2 при росте x на 1. При x=1 → y=3.
Формула: y = 2x + 1
Проверка: 2·1+1=3, 2·2+1=5, 2·3+1=7. Всё верно!

📊 Координатная плоскость

График показывает зависимость визуально. По горизонтали — x, по вертикали — y.

Точка (2; 5) означает: x = 2, y = 5.
Точка (0; 3) — начало отсчёта по y при x = 0.
Сначала идём вправо/влево по оси x, потом вверх/вниз по оси y.

📈 Построение графика по таблице

Нанесите точки из таблицы на координатную плоскость и соедините их.

y = x + 2
x0123
y2345
Точки: (0;2), (1;3), (2;4), (3;5) — лежат на одной прямой.

🔄 Обратная зависимость

Иногда y уменьшается при росте x. Тогда коэффициент перед x отрицательный.

y = 12 − x
x2345
y10987
y = 8 − x — тоже убывающая зависимость. При x=0 → y=8.

🛍️ Формула стоимости

Цена = количество × цена за штуку. Это классическая зависимость y = k·x.

Майки по 10 ₼: y = 10x
Тетради по 0,2 ₼: y = 0,2x
При x = 3 майках: y = 10·3 = 30 ₼

🎯 Проверка себя

Чтобы убедиться, что формула верна, подставьте 2–3 значения из таблицы.

Формула y = 4x − 1
x=1 → 3 ✓    x=2 → 7 ✓    x=3 → 11 ✓
Если хотя бы одно значение не совпадает — формула неверна.
🪜 Пошаговые алгоритмы
1

Определи переменные

Найди, какая величина меняется сама по себе (независимая x), а какая от неё зависит (зависимая y).

2

Запиши формулу или правило

Если дана машина — переведи её действия в формулу. Если дана таблица — найди закономерность.

3

Составь таблицу значений

Подставь заданные значения x в формулу и вычисли y. Проверь порядок действий.

4

Построй график (если требуется)

Нанеси точки (x; y) на координатную плоскость. Отметь ось x вправо, ось y вверх. Соединяй точки линией.

5

Проверь ответ

Подставь любое значение x обратно в формулу. Убедись, что таблица и график совпадают.

📝 Разобранные примеры
№ 1 Формула периметра квадрата
Сторона квадрата a = 7 см. Найдите периметр P.
✔ Формула: P = 4a
✔ P = 4 · 7 = 28 см
№ 2 Площадь прямоугольника
Длина a = 8 см, ширина b = 5 см. Найдите площадь S.
✔ Формула: S = a · b
✔ S = 8 · 5 = 40 см²
№ 3 Вычислительная машина
Машина: x → ×2 → +1 → y. Найдите y при x = 4 и x = 6.
✔ Формула: y = 2x + 1
✔ При x = 4: y = 2·4 + 1 = 9
✔ При x = 6: y = 2·6 + 1 = 13
№ 4 Найди формулу по таблице
x1234
y581114
Найдите формулу зависимости y от x.
✔ Разность при росте x на 1: y растёт на 3. Значит, есть 3x.
✔ При x = 1: 3·1 = 3, но y = 5. Не хватает 2.
✔ Формула: y = 3x + 2
№ 5 Заполнение таблицы
Заполните таблицу для y = 12 − x при x = 2, 3, 4, 5, 6.
✔ y = 12 − 2 = 10
✔ y = 12 − 3 = 9
✔ y = 12 − 4 = 8
✔ y = 12 − 5 = 7
✔ y = 12 − 6 = 6
✅ Таблица: 10, 9, 8, 7, 6
№ 6 График по точкам
Постройте график y = x + 2, используя точки (0;2), (1;3), (2;4), (3;5).
✔ Отметьте ось x (вправо) и ось y (вверх).
✔ Нанесите точки: (0;2) — на оси y, (1;3) — вправо 1, вверх 3 и т.д.
✔ Соедините точки прямой линией. Получится возрастающий график.
№ 7 Формула пути
Скорость v = 60 км/ч, время t = 3 ч. Найдите путь S.
✔ Формула: S = v · t
✔ S = 60 · 3 = 180 км
№ 8 Стоимость покупки
n тетрадей по 0,5 ₼ и m ручек по 1,2 ₼. Найдите стоимость при n = 4, m = 3.
✔ Формула: y = 0,5n + 1,2m
✔ y = 0,5·4 + 1,2·3 = 2 + 3,6 = 5,6 ₼
