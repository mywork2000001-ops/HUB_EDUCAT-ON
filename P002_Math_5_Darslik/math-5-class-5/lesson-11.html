🗺️ Структура раздела 5

Раздел объединяет пять связанных тем в одну систему. Всё начинается с переменной и заканчивается графиком.

Переменная → Выражение → Уравнение/Неравенство → Зависимость → График
Каждая следующая тема опирается на предыдущую. Нельзя понять график, не зная переменную.

🔢 Выражения с переменной

Переменная заменяет число. Сумму одинаковых слагаемых записываем как произведение.

5·a = a + a + a + a + a
x + x + x = 3x
Подстановка: при a=4 → 5·4 = 20
Порядок действий: скобки → умножение/деление → сложение/вычитание.

⚖️ Уравнения и свойства равенств

Уравнение — равенство с переменной. Решаем с помощью обратных действий.

3x + 2 = 8
3x = 8 − 2 → 3x = 6 → x = 2
Свойства: a=b ↔ a+c=b+c; a=b ↔ a·c=b·c (c≠0). Можно переносить с изменением знака.

📐 Неравенства

Неравенство — сравнение двух выражений. Знаки: >, <, ≥, ≤.

x + 3 < 9 → x < 6
2y ≥ 10 → y ≥ 5
5 − x > 2 → x < 3
На числовой оси: > и < — выколотая точка, ≥ и ≤ — закрашенная. Стрелка вправо — больше, влево — меньше.

🔗 Зависимость между переменными

Если от изменения x меняется y — это зависимость. x — независимая, y — зависимая переменная.

y = x + 4
x=0 → y=4; x=1 → y=5; x=2 → y=6
Зависимость можно задать формулой, таблицей или графиком.

⚙️ Вычислительные машины

Машина преобразует вход x в выход y по алгоритму.

x → +3 → y: формула y = x + 3
x → ×2 → −1 → y: формула y = 2x − 1
x=5 → y = 2·5 − 1 = 9

📋 Таблицы зависимостей

Таблица показывает соответствие x → y. Помогает найти формулу и построить график.

y = 2x + 1
x01234
y13579

📈 Координатная плоскость и графики

Горизонтальная ось — x (абсцисса), вертикальная — y (ордината). Точка (x;y).

y = x + 2: точки (0;2), (1;3), (2;4), (3;5)
Нанесите → соедините → получите прямую
Сначала вправо/влево по x, потом вверх/вниз по y.

📐 Формулы из геометрии и жизни

Формулы — это тоже зависимости между величинами.

P = 4a (периметр квадрата)
S = a·b (площадь прямоугольника)
S = v·t (путь = скорость × время)
y = k·x (цена = количество × цена за штуку)

🔍 Нахождение формулы по таблице

Смотрим на разность между соседними y при росте x на 1.

x: 1, 2, 3, 4 → y: 5, 8, 11, 14
Разность = 3. При x=1 → 3·1=3, но y=5. Не хватает 2.
Формула: y = 3x + 2
Проверьте: 3·1+2=5 ✓, 3·2+2=8 ✓, 3·3+2=11 ✓

🔄 Обратные зависимости

Иногда y уменьшается при росте x. Коэффициент при x отрицательный или формула вычитания.

y = 12 − x: при x=2 → y=10; x=5 → y=7
y = 20 − 3x: при x=1 → y=17; x=2 → y=14
График убывающей зависимости идёт вниз при движении вправо.

🎯 Система знаний

Все темы связаны. Переменная входит в выражение, выражение — в уравнение, уравнение — в формулу зависимости, формула — в таблицу и график.

a + a + a = 3a → 3a + 2 = 14 → a = 4
y = 3x + 2 → таблица → график
P = 4a → при a=5 → P=20
Умение переводить задачу из одной формы в другую — ключевой навык раздела 5.

📝 Разобранные примеры — комплексные задачи
№ 1 От выражения к уравнению
Сумма пяти одинаковых чисел равна 35. Составьте уравнение и найдите число.
✔ Пусть число = x. Пять одинаковых → 5x.
✔ Уравнение: 5x = 35
✔ x = 35 : 5 = 7
✔ Проверка: 5·7 = 35 ✓
№ 2 Уравнение из текста
Произведение числа и 3 равно сумме 15 и 6. Найдите число.
✔ «Произведение числа и 3» → 3x
✔ «Сумма 15 и 6» → 15 + 6 = 21
✔ Уравнение: 3x = 21 → x = 7
№ 3 Неравенство и числовая ось
Решите неравенство 2x + 4 ≤ 14. Изобразите решение на числовой оси.
✔ 2x ≤ 14 − 4 → 2x ≤ 10
✔ x ≤ 5
✔ На числовой оси: закрашенная точка на 5, стрелка влево.
✅ Подходят: 5, 4, 3, 2, 1, 0, −1…
№ 4 Вычислительная машина → формула → таблица
Машина: x → ×3 → −2 → y. Составьте формулу, заполните таблицу для x = 1, 2, 3, 4, 5.
✔ Формула: y = 3x − 2
✔ x=1 → y=1; x=2 → y=4; x=3 → y=7; x=4 → y=10; x=5 → y=13
✔ Таблица: 1, 4, 7, 10, 13 (разность = 3)
№ 5 Нахождение формулы по таблице → график
x1234
y6101418
Найдите формулу, постройте график.
✔ Разность = 4. При x=1: 4·1=4, но y=6. Не хватает 2.
✔ Формула: y = 4x + 2
✔ Точки: (1;6), (2;10), (3;14), (4;18) — лежат на одной прямой.
№ 6 Геометрия + формула зависимости
Периметр прямоугольника P = 2(a + b). При a = 5 см найдите P для b = 2, 3, 4, 5 см. Постройте график P(b).
✔ P = 2(5 + b) = 10 + 2b
✔ b=2 → P=14; b=3 → P=16; b=4 → P=18; b=5 → P=20
✔ График: точки (2;14), (3;16), (4;18), (5;20) — возрастающая прямая.
№ 7 Текстовая задача на зависимость
За 1 час автобус проходит 60 км. Составьте формулу пути S(t). Найдите S за 2, 3, 4 часа. Постройте график.
✔ Формула: S = 60t
✔ t=2 → S=120; t=3 → S=180; t=4 → S=240
✔ График: (2;120), (3;180), (4;240) — прямая через начало координат.
№ 8 Комплексная задача: всё вместе
Магазин продаёт ручки по x манат и тетради по y манат. a) Составьте формулу стоимости C при покупке 3 ручек и 5 тетрадей. b) Найдите C при x=1,2, y=0,5. c) Решите неравенство C < 10.
✔ a) C = 3x + 5y
✔ b) C = 3·1,2 + 5·0,5 = 3,6 + 2,5 = 6,1 ман.
✔ c) 3x + 5y < 10. При y=0,5: 3x + 2,5 < 10 → 3x < 7,5 → x < 2,5
✅ Ручки должны стоить меньше 2,5 ман.
