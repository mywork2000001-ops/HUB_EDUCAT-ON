🪜Глава 2. Пошаговые алгоритмы

Пятиступенчатый алгоритм для любого линейного уравнения и задачи на составление уравнения.

1

Раскройте скобки и упростите

Если в уравнении есть скобки — умножьте множитель на каждое слагаемое внутри. Приведите подобные слагаемые в каждой части уравнения отдельно.

2

Перенесите слагаемые с переменной влево, числа — вправо

Всё с x отправляйте в левую часть, всё без x — в правую. Не забудьте: при переносе знак меняется на противоположный.

3

Приведите подобные слагаемые

Сложите коэффициенты при x слева. Сложите все числа справа. Получите уравнение вида: ax = b.

4

Разделите правую часть на коэффициент при x

x = b : a. Если a = 0 и b ≠ 0 — корней нет. Если a = 0 и b = 0 — x любое число (бесконечно много корней).

5

Проверьте ответ подстановкой

Подставьте найденное значение x в исходное уравнение. Вычислите левую и правую части отдельно. Если равны — решение верно.
