Сложение смешанных чисел

Урок 5. Сложение смешанных чисел с одинаковыми и разными знаменателями. 20 вопросов.

Что вы изучите

Смешанное = целое + дробь

2¾ = 2 + ¾. Складываем части отдельно.

Сложение (одинаковые знаменатели)

Складываем целые отдельно, дробные отдельно.

Перенос дробной части

Разные знаменатели

Визуальная теория — 12 слайдов

Листайте слайды. Каждый блок — ключевой элемент темы.

Слайд 1 из 12

Смешанное число = целое + дробь

Смешанное число состоит из целой части и правильной дробной части.

2¾ = 2 + ¾

Примеры:

1½ = 1 + 1/2
3⅔ = 3 + 2/3
5⅜ = 5 + 3/8
При сложении можно складывать целые и дробные части отдельно, затем объединять.

Сложение: складываем целые и дробные части отдельно

Метод: целое + целое = целое, дробь + дробь = дробь.

2⅓ + 1⅓ = (2+1) + (1/3+1/3) = 3 + 2/3 = 3⅔
Работает когда знаменатели дробных частей одинаковы.

Сложение с одинаковыми знаменателями

Складываем целые части, складываем дробные части.

3⅖ + 2⅕ = (3+2) + (2/5+1/5) = 5 + 3/5 = 5⅗
1¾ + 2¾ = (1+2) + (3/4+3/4) = 3 + 6/4 = 3 + 1½ = 4½
Если сумма дробных частей ≥ 1 — прибавить 1 к целой части!

Сложение с разными знаменателями (через НОК)

Привести дробные части к общему знаменателю (НОК), затем сложить.

1½ + 2⅓:
НОК(2,3)=6
1 3/6 + 2 2/6 = 3 5/6
Изменяем только дробные части. Целые части не трогаем до последнего шага.

Перенос: если дробная часть ≥ 1

Если сумма дробных частей неправильная дробь — выделяем целую часть.

2⅔ + 1¾:
НОК(3,4)=12
2 8/12 + 1 9/12 = 3 17/12 = 3 + 1 5/12 = 4 5/12
17/12 = 1 целое + 5/12. Прибавляем 1 к целой части: 3+1=4.

Упрощение результата

После сложения всегда проверяем: можно ли сократить дробную часть?

Алгоритм:

Сложить как обычно
Если дробная часть ≥ 1 — выделить целую часть
Сократить дробную часть (найти НОД)
Записать ответ в виде смешанного числа
Ответ должен быть в виде несократимой дроби.

Сложение трёх и более смешанных чисел

Складываем по очереди или находим НОК всех знаменателей сразу.

1¼ + 1½ + 1¾:
НОК(4,2,4)=4
1 1/4 + 1 2/4 + 1 3/4 = 3 6/4 = 3 + 1½ = 4½

Выражения со смешанными числами

Порядок действий такой же, как с целыми числами: скобки → умножение → сложение.

(1½ + 2¼) + 1⅓:
= 3¾ + 1⅓
НОК(4,3)=12: 3 9/12 + 1 4/12 = 4 13/12 = 5 1/12

Перевод в неправильную дробь перед сложением

Альтернативный метод: перевести оба смешанных в неправильные дроби, сложить, перевести обратно.

2½ + 1⅓:
5/2 + 4/3
НОК=6: 15/6 + 8/6 = 23/6 = 3⅚
Оба метода дают одинаковый результат. Выбирайте удобный.

Задачи на суммирование смешанных чисел

Примеры:

Длина пути: 2½ км + 1¾ км = 4¼ км
Вес: 1⅓ кг + 2⅔ кг = 4 кг
Время: 1½ ч + 2⅔ ч = 4⅙ ч
Сначала определите единицы, потом складывайте.

Проверка результата

Чтобы проверить сложение — вычитаем одно слагаемое из суммы.

2½ + 1⅓ = 3⅚
Проверка: 3⅚ - 1⅓ = 3⅚ - 1 2/6 = 3 5/6 - 1 2/6 = 2 3/6 = 2½ ✓
