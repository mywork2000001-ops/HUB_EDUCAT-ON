Сложение и вычитание дробей с разными знаменателями

Урок 4. Нахождение НОК, приведение к общему знаменателю, сложение и вычитание. 20 вопросов.

Что вы изучите

Сложение (одинаковые)

a/c + b/c = (a+b)/c — складываем только числители.

НОК — основа

Нахождение наименьшего общего кратного для разных знаменателей.

Сложение (разные)

Привести к НОК, пересчитать, сложить, сократить.

Вычитание

Тот же алгоритм, но вычитаем числители.

Визуальная теория — 12 слайдов

Листайте слайды. Каждый блок — ключевой элемент темы.

Слайд 1 из 12

Сложение с одинаковым знаменателем

Если знаменатели равны — складываем числители, знаменатель не меняется.

a/c + b/c = (a+b)/c

Примеры:

3/7 + 2/7 = 5/7
4/9 + 5/9 = 9/9 = 1
1/5 + 3/5 = 4/5
Нельзя складывать знаменатели! 2/5+1/5 = 3/5, НЕ 3/10.

Сложение с разными знаменателями: шаг 1 — найти НОК

Сначала находим НОК знаменателей.

Сложить 1/3 + 1/4
Кратные 3: 3, 6, 9, 12...
Кратные 4: 4, 8, 12, 16...
НОК(3,4) = 12
НОК — наименьшее число, которое делится на оба знаменателя.

Сложение: шаг 2 — умножить числители

Приводим каждую дробь к знаменателю = НОК.

1/3 → ×4 → 4/12
1/4 → ×3 → 3/12

Множитель = НОК ÷ знаменатель дроби.

Числитель умножаем на то же число, что и знаменатель.

Сложение: шаг 3 — сложить и сократить

Теперь складываем числители и при необходимости сокращаем.

4/12 + 3/12 = 7/12

Если результат неправильная дробь — переводим в смешанное число.

Проверьте: можно ли сократить результат? НОД(7,12)=1 — уже несократима.

Вычитание с одинаковым знаменателем

Вычитаем только числители, знаменатель тот же.

a/c - b/c = (a-b)/c

Примеры:

7/9 - 4/9 = 3/9 = 1/3
5/6 - 1/6 = 4/6 = 2/3
1 - 3/8 = 8/8 - 3/8 = 5/8
Если числитель результата = 0, дробь равна 0. Нельзя получить отрицательный числитель (в 5 классе).

Вычитание с разными знаменателями

Алгоритм тот же: НОК → привести → вычесть числители → сократить.

3/4 - 1/6:
НОК(4,6)=12
9/12 - 2/12 = 7/12
Три шага: НОК → привести → вычесть. Не забыть сократить!

Проверка результата (обратное действие)

Чтобы проверить сложение — вычитаем, чтобы проверить вычитание — складываем.

Пример проверки:

3/4 - 1/6 = 7/12
Проверка: 7/12 + 1/6 = 7/12 + 2/12 = 9/12 = 3/4 ✓
Всегда проверяйте: если вернули исходное число — ответ верен.

Сложение трёх дробей

Находим НОК всех трёх знаменателей и приводим каждую дробь.

1/2 + 1/3 + 1/6:
НОК(2,3,6)=6
3/6 + 2/6 + 1/6 = 6/6 = 1
НОК нескольких чисел: НОК(a,b,c) = НОК(НОК(a,b),c).

Выражения с несколькими дробями

В выражениях соблюдаем порядок действий: скобки → умножение/деление → сложение/вычитание.

5/6 - 1/4 + 1/3:
НОК(6,4,3)=12
10/12 - 3/12 + 4/12 = 11/12
Если нет скобок, вычисляем слева направо.

Частный случай: дробь и целое число

Целое число записываем как дробь с знаменателем 1, затем приводим к НОК.

2 + 3/4:
2 = 8/4
8/4 + 3/4 = 11/4 = 2¾
3 - 5/6:
3 = 18/6
18/6 - 5/6 = 13/6 = 2⅙

Задачи на сложение/вычитание дробей

Типовые задачи:

В тарелке 3/4 торта. Съели 1/6. Осталось: 3/4-1/6=9/12-2/12=7/12
Было 5/8 кг муки, добавили 1/4 кг: 5/8+2/8=7/8 кг
Из 1 ч. сделали: 1/3 ч. работы + 1/4 ч. отдыха = 7/12 ч.

Итог урока

1Одинаковые знаменатели: складываем/вычитаем числители
2Разные знаменатели: НОК → привести → сложить/вычесть → сократить
3Три шага: НОК, пересчёт числителей, действие
4Проверка: обратное действие
5Целое + дробь: записать целое как дробь
