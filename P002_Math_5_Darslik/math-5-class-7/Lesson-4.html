Площадь поверхности прямой призмы

Урок 4. §7.2. Призма с прямоугольным треугольником в основании. S = 2·S_осн + S_бок. 30 вопросов для полного освоения темы.

🎯Что вы изучите

Прямая призма

Тело с основаниями-равными многоугольниками и перпендикулярными боковыми рёбрами. Боковые грани — прямоугольники.

Прямоугольный треугольник в основании

5 граней, 9 рёбер, 6 вершин. Катеты a, b; гипотенуза c = √(a²+b²).

Формула поверхности

S = 2·S_осн + S_бок = a·b + (a+b+c)·h. Основание + боковая поверхность.

Боковая поверхность

S_бок = P · h, где P = a+b+c — периметр основания, h — высота призмы.

🪜Глава 2. Алгоритм вычисления площади поверхности прямой призмы

Пошаговый алгоритм для нахождения полной поверхности треугольной прямой призмы.

1

Определи катеты и гипотенузу основания

Запиши катеты a и b прямоугольного треугольника в основании. Найди гипотенузу: c = √(a² + b²). Например: a=3, b=4 → c=√(9+16)=√25=5.

2

Найди площадь основания

Площадь прямоугольного треугольника: S_осн = ½ × a × b. Две основания дают: 2 × S_осн = a × b. Пример: a=3, b=4 → 2·S_осн = 3×4 = 12 см².

3

Найди периметр основания

Периметр треугольного основания: P = a + b + c. Пример: P = 3+4+5 = 12 см. Это нужно для расчёта боковой поверхности.

4

Вычисли боковую поверхность

S_бок = P × h, где h — высота призмы. Боковая поверхность = три прямоугольника: a×h + b×h + c×h = (a+b+c)×h. Пример: P=12, h=6 → S_бок=72 см².

5

Сложи: S = 2·S_осн + S_бок

Полная поверхность = два основания + боковая поверхность. S = a·b + (a+b+c)·h. Запиши единицы (см², м²). Пример: S = 12 + 72 = 84 см².
